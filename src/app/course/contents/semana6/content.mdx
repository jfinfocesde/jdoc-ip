## 2. Crea una página web sencilla con HTML (40%)

**Instrucciones:**

### **Planificación:**

-   **Tema:** Elige un tema que te interese (deportes, música, viajes, etc.) para la página web.
-   **Contenido:** Define qué información quieres mostrar en la página web (texto, imágenes).

### **Desarrollo:**

-   **Creación del archivo HTML:** Crea un nuevo archivo con la extensión `.html`.
-   **Estructura básica:** Define la estructura básica de la página web con las etiquetas `<!DOCTYPE html>`, `<html>`, `<head>`, `<head>` , `<body>` y , `<footer>`.
-   **Contenido:** Agrega el contenido a la sección `<body>` utilizando las etiquetas HTML:
    -   `<h1>` a `<h6>` para encabezados.
    -   `<p>` para párrafos.
    -   `<br>` para saltos de línea.
    -   `<img>` para imágenes.
    -   `<ul>` y `<ol>` para listas.        

### **Subir a Github:**

-   Crea un nuevo repositorio en Github 
-   Subir el proyecto al repositorio.
-   Publica el repositorio en github pages.
