# Actividad 3: Diseñar un formulario en HTML

## Objetivo:

El objetivo de esta actividad es que los estudiantes practiquen la creación de formularios HTML utilizando los diversos atributos disponibles para los campos de formulario.

Instrucciones:

1.  Crea un nuevo archivo HTML en tu editor de código.

2.  En el archivo HTML, crea un formulario con los siguientes campos:

    -   Nombre: Campo de tipo texto (type="text")
    -   Apellido: Campo de tipo texto (type="text")
    -   Correo electrónico: Campo de tipo email (type="email")
    -   Edad: Campo de tipo número (type="number") con un valor mínimo de 18 y un valor máximo de 100
    -   Fecha de nacimiento: Campo de tipo fecha (type="date")
    -   Género: Campo de tipo radio (type="radio") con dos opciones: "Hombre" y "Mujer"
    -   Comentario: Campo de tipo textarea (type="textarea")
3.  Agrega los siguientes atributos a cada campo:

    -   Nombre: `required`
    -   Apellido: `required`
    -   Correo electrónico: `required` 
    -   Edad: `required`
    -   Fecha de nacimiento: `required`
    -   Género: `required`
    -   Comentario: `placeholder="Ingrese su comentario aquí"`
4.  Agrega una etiqueta (`label`) para cada campo. La etiqueta debe describir claramente el propósito del campo.

5.  Agrega un botón de envío (`input type="submit"`) al formulario.

6.  Agrega en el atributo action del formulario(`action="https://echo.free.beeceptor.com"`).

7.  Guarda el archivo HTML y ábrelo en tu navegador web.

8.  Completa el formulario con tu información personal y envíalo.

## Nota

`https://echo.free.beeceptor.com` es un servicio en línea gratuito proporcionado por Beeceptor que te permite simular un servidor de eco HTTP. Un servidor de eco HTTP es un servidor simple que responde a cualquier solicitud HTTP que recibe devolviendo exactamente los mismos datos que se le enviaron.
